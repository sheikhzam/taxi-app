Have nodejs and npm installed

clone the app
open the app folder in terminal
cd backend
npm i
create a .env file
add the following environment variables
  - PORT
  - MONGODB_URI
  - JWT_SECRET 
  - JWT_EXPIRES_IN
npm start


in a separate terminal open the same app folder
in a separate terminal
cd frontend
npm i
npm run dev
- Frontend will run in port 3000
